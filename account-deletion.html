1. 삭제 요청 방법

방법 A. 앱 내 회원 탈퇴 (권장)

  1. Chromission 앱 실행
  2. 마이페이지 → 설정 → 회원 탈퇴 메뉴 선택
  3. 안내 화면을 확인한 뒤 탈퇴 확인

앱 내 회원 탈퇴 기능은 출시 시점에 제공됩니다. 그 전에는 아래 이메일을 이용해 주세요.

방법 B. 이메일로 요청

  1. haghl980813@gmail.com 으로 메일 발송
  2. 제목: [Chromission] 계정 삭제 요청
  3. 본문에 다음 정보를 기재
    • 가입에 사용한 로그인 이메일 (Google 또는 Apple)
    • 삭제 사유 (선택, 답변 필수 아님)

요청 접수 후 본인 확인을 거쳐 회원 탈퇴를 처리하며, 처리 완료 시 회신 메일로 안내드립니다.

2. 삭제되는 데이터

구분 처리
회원 정보 (이메일, 닉네임, 소셜 식별자, 프로필 이미지) 요청 즉시 비활성화 후 30일 이내 영구 삭제
업로드한 사진 파일 및 그리드 요청 즉시 비공개 처리 후 30일 이내 저장소에서 영구 삭제
일일 미션 기록, 그리드 메타정보 요청 즉시 비활성화 후 30일 이내 영구 삭제
좋아요·팔로우 등 커뮤니티 활동 기록 (해당 시) 요청 즉시 영구 삭제
리프레시 토큰 요청 즉시 영구 삭제 (모든 기기 자동 로그아웃)
광고 식별자(IDFA · GAID) 요청 즉시 처리 종료 (서버 보유분 폐기)

3. 일정 기간 보관되는 데이터

관련 법령에 따라 회원 탈퇴 후에도 일정 기간 분리 보관되는 항목입니다.

항목 보관 기간 근거 법령
서비스 이용 기록 (접속 로그, IP 등) 3개월 「통신비밀보호법」
고객 문의 기록 3년 「전자상거래법」

위 데이터는 회원 식별이 불가능한 형태(또는 분리 보관)로 처리되며, 보관 기간이 지나면 즉시 영구 삭제됩니다.

4. 일부 데이터만 삭제하기

회원 탈퇴 없이도 다음과 같이 데이터의 일부만 선택적으로 삭제할 수 있습니다.

  • 개별 사진 삭제: 그리드 편집 화면에서 사진 한 장 단위로 삭제할 수 있습니다.
  • 그리드 단위 삭제: 마이페이지 → MY POST에서 특정 날짜 그리드를 삭제할 수 있습니다. (출시 시점 제공)
  • 이메일 요청: 위 1번의 이메일 방법으로 "특정 데이터 삭제 요청"을 보내주시면 본인 확인 후 처리해 드립니다.

5. 처리 기한

  • 앱 내 탈퇴: 즉시 처리 (계정 비활성화). 사진 등 부속 데이터는 백업 정책에 따라 최대 30일 이내 영구 삭제.
  • 이메일 요청: 영업일 기준 7일 이내 처리. 처리 완료 시 회신 메일 발송.
